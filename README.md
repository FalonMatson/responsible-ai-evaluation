# Responsible AI Evaluation Toolkit

A toolkit designed to measure bias, fairness, and explainability in machine learning models.

## Features

- Bias detection metrics
- Fairness evaluation
- Model explainability tools
- Performance monitoring

## Components

src/bias_metrics.py – fairness metrics  
src/explainability.py – feature importance tools  
src/evaluation.py – model evaluation pipeline  

## Supported Models

- Scikit-learn models
- PyTorch models
- TensorFlow models

## Goal

To support responsible deployment of machine learning systems in enterprise environments.
